## 5. 项目文件结构（当前）

```
/workspace/B-BABO-game/
├── index.html                          ← 入口 HTML
├── package.json                        ← 项目配置
├── tsconfig.json                       ← TS 配置
├── vite.config.ts                      ← Vite 配置（路径别名）
├── .gitignore
├── README.md
├── docs/
│   ├── 开发文档_v2.0.md                ← 开发文档副本
│   └── 开发上下文记录.md               ← 上下文记录副本
└── src/
    ├── main.ts                         ← 入口（DOMContentLoaded启动Game）
    ├── core/
    │   └── Game.ts                     ← 游戏核心类（生命周期/角色切换/resize）
    ├── engine/
    │   ├── GameLoop.ts                ← 固定时间步长游戏循环
    │   ├── Renderer.ts                ← Canvas 2D 渲染器 + HUD
    │   ├── Camera.ts                  ← 2D 摄像机（平滑跟随）
    │   ├── InputManager.ts            ← 输入管理（键盘+鼠标+触摸+tap）
    │   ├── EntityManager.ts           ← 实体管理器（简化ECS）
    │   └── PixelRenderer.ts           ← 像素精灵渲染器（20键调色板系统）
    ├── entities/
    │   ├── BaboSprite.ts              ← HIRONO 基础精灵模板（32×32）
    │   └── PlayerFactory.ts           ← 角色工厂（旧版，已被CharacterRegistry替代）
    ├── systems/
    │   ├── MovementSystem.ts          ← 移动系统（WASD/方向键）
    │   └── RenderSystem.ts            ← 渲染系统（视锥剔除+网格背景）
    └── data/
        ├── CharacterRegistry.ts       ← 20角色注册表（属性/技能/精灵/调色板）
        ├── sprites_batch1.ts          ← 角色1-5精灵（战士/法师/游侠/守卫/烈焰）
        ├── sprites_batch2.ts          ← 角色6-10精灵（死灵/雷电/冰霜/幻影/商人）
        ├── sprites_batch3.ts          ← 角色11-15精灵（德鲁伊/机甲/吟游/狼人/幽灵）
        └── sprites_batch4.ts          ← 角色16-20精灵（星辰/赌徒/毒菇/时空/龙裔）
```

### 10.2 开发工作流

```
1. 读取 /workspace/开发上下文记录.md → 了解当前进度
2. 读取开发文档相关章节 → 了解详细设计
3. 复杂任务调用 brainstorming skill → 规划方案
4. 多步骤任务调用 writing-plans skill → 编写计划
5. 独立子任务 → 启动子代理并行开发
6. 执行开发 → 编写代码
7. agent-browser 验证 → 截图确认
8. 更新上下文记录 → 记录进度
9. git push → 推送 GitHub
```

### 10.3 代码质量规则

- TypeScript 严格类型
- ECS 架构模式
- JSDoc 注释
- ObjectPool 管理频繁创建对象

| 素材类型 | 方案 | 尺寸 |
|----------|------|------|

### 10.5 HIRONO 角色美术规范

- 20 个角色对应不同 HIRONO 系列造型
- 核心特征：凌乱碎短发、浮肿大眼皮、红鼻尖、腮红、宽松外套
- 调色板：20 语义化颜色键（hair/skin/eyeW/eyeI/eyeB/lid/lidSh/blush/nose/mouth/coat/coatHi/coatDk/coatLn/inner/shoe/outline/white + 角色特有键）
- 色调：低饱和度柔和色调，每个角色独特配色
- 幻影角色特殊处理：直接复用 HIRONO_BASE_SPRITE + 独立调色板
